# 📈 AISA (AI-driven ISA Portfolio Manager)

> **"낮은 변동성, 안정적인 수익률"**을 위한 개인 맞춤형 ISA 포트폴리오 관리 서비스

## 📖 프로젝트 소개

재테크 열풍 속에서 많은 개인 투자자들이 주식 시장에 뛰어들고 있습니다. 하지만 대부분의 사람들이 어떤 종목과 상품에 얼마나 투자하는 것이 낮은 변동성을 가지면서 안정적인 수익률을 가질 수 있을지에 대한 고민이 많아 보입니다.
이러한 문제를 해결하기 위해 **개인 맞춤형 ISA 포트폴리오**를 설계하고 관리해주는 웹 서비스를 개발하고자 합니다.
단순한 종목 추천을 넘어, **변동성을 낮추고 안정적인 수익**을 낼 수 있는 자산 배분 전략을 제안합니다.

## 🚀 핵심 기능 계획

### 1. 📊 시장 데이터 분석 (Market Data)

- **실시간/과거 데이터 조회**: 국내/해외 주식, 주요 지수(KOSPI, S&P500 등), 환율, 경제 지표 조회
- **KIS(한국투자증권) API 연동**: 신뢰할 수 있는 실시간 금융 데이터 제공
- **하이브리드 지수 차트**: 실시간 데이터(KIS)와 과거 데이터(DB)를 결합하여 최적의 차트 데이터 제공
- **배당금 조회 (OpenDart)**: 기업의 과거 배당금 지급 내역 및 수익률 조회

### 2. 💼 포트폴리오 관리 (Portfolio Management)

- **나만의 포트폴리오 구성**: 관심 종목으로 포트폴리오를 구성하고 관리
- **포트폴리오 진단**:
    - **자산 상관관계 분석**: 포트폴리오 내 자산 간의 상관계수(Pearson) 분석 및 시각화 (히트맵)
    - **변동성 분석**: 시장 지수 대비 포트폴리오의 변동성 비교
- **백테스팅 (Backtesting)**: 구성한 포트폴리오의 과거 성과(수익률, MDD, Sharpe Ratio) 시뮬레이션

### 3. 💰 ISA 특화 기능 (ISA Specialization)

- **절세 효과 시뮬레이션**: 일반 계좌 대비 ISA 계좌 활용 시 예상 세금 절감액 계산
- **납입 한도 관리**: 연간 납입 한도 및 만기 관리 대시보드

### 4. 🤖 AI 투자 비서 (AI Chatbot)

- **RAG 기반 챗봇**: 사용자의 포트폴리오 데이터를 기반으로 한 맞춤형 투자 상담
- **투자 성향 진단**: 사용자 성향(공격형/안정형 등)에 따른 포트폴리오 비중 제안
- **리밸런싱 알림**: 목표 비중 이탈 시 리밸런싱 필요성 진단 및 알림 (대시보드)
